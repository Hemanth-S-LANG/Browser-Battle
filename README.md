RNSIT — RNS Institute of Technology Website

A modern, fully responsive multi-page college website built with React, Tailwind CSS, and Framer Motion.

Tech Stack

React 19 + Vite 8
Tailwind CSS v4
Framer Motion (animations)
React Router DOM v7
Lucide React (icons)
Pages (13 total)

Home, About, Departments, Department Detail, Academics, Admissions, Placements, Research, Campus Life, Events, Alumni, Students, Contact
Key Features

Animated particle star field canvas (adapts to light/dark mode)
Hero section with cycling word animation and floating orbs
Animated counters for stats (count-up on scroll)
Department listing with search/filter and individual detail pages per department
Faculty cards, lab listings, achievements, and image gallery per department
Events page with search + category + upcoming/past filters
Placements page with year-wise stats table and recruiter grid
Contact page with form validation and Google Maps embed
Campus Life photo gallery with lightbox
Light/Dark mode toggle (persisted in localStorage)
Scroll-triggered animations on all sections
Sticky navbar with dropdown menu and active route indicator
Fully responsive — mobile, tablet, desktop
CSS variable-based theming for clean mode switching
SEO-friendly page titles and meta descriptions
